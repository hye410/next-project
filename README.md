  ## Skill
    - 게시글 좌측 상단에 한 페이지당 보이는 게시글의 개수를 보여줍니다.
    - 품종을 선택하면 상세 페이지로 이동합니다.
    - 상세 페이지에서 HOME 버튼을 클릭하면 다시 HOME으로 돌아옵니다.
    - 정렬 버튼을 클릭하여 게시글을 새롭게 정렬할 수 있습니다.
    - pagination이 적용돼있어 하단에 페이지 버튼을 클릭하면 다음 페이지로 이동합니다.
